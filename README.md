# RAG Document Processor Doc-Proc

## Doc-Proc is a document processing application that leverages Retrieval-Augmented Generation (RAG) techniques to extract and process information from various document formats, primarily PDFs. The application allows users to upload documents, extract text, and generate responses based on the content using a vector database and embedding models.

## Description

This project is a document processing application that utilizes Retrieval-Augmented Generation (RAG) techniques to extract and process information from various document formats. The application is designed to handle PDF files, allowing users to upload documents, extract text, and generate responses based on the content.

## Features

- **Document Upload**: Users can upload PDF documents for processing.
- **Text Extraction**: The application extracts text from the uploaded PDF files.
- **Chunking**: The extracted text is divided into manageable chunks for processing.
- **Vector Database**: The application uses a vector database to store and retrieve information efficiently.
- **Embedding Model**: Utilizes a pre-trained embedding model to convert text into vector representations for better retrieval performance.
